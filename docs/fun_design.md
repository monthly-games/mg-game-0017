## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 재료 수집/가공                                       │
│         ↓                                               │
│  2. 레시피 연구/제작                                     │
│         ↓                                               │
│  3. 아이템 판매/납품                                     │
│         ↓                                               │
│  4. 공방 확장/시설 업그레이드 ─────────────→ (1로 반복)   │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **재료 수집**: 던전 탐사, 채집, 구매로 재료 확보
2. **레시피 연구/제작**: 새 레시피 발견, 아이템 제작
3. **판매/납품**: 모험가에게 판매, 던전 길드 납품
4. **공방 확장**: 시설 업그레이드, 장인 고용

## 3. 재미 축 (Fun Pillars)

### 3.1 제작/설계 (Crafting)
- **목표**: 복잡한 제작 시스템을 마스터하는 재미
- **핵심 경험**: 전설 아이템을 직접 만들었을 때의 성취감
- **주요 요소**:
  - 레시피 계보 시스템
  - 품질/등급 시스템
  - 특수 효과 부여
  - 마스터 제작

### 3.2 경제/수요 대응 (Economy)
- **목표**: 시장 수요에 맞춰 생산하는 전략적 재미
- **핵심 경험**: 트렌드를 읽고 대박 났을 때의 쾌감
- **주요 요소**:
  - 수요/공급 시스템
  - 주문 시스템
  - 시즌 이벤트
  - 경쟁 공방

### 3.3 파생 시스템 (Meta Systems)
- **목표**: 단순 제작을 넘어선 복합 시스템 운영
- **핵심 경험**: 내 공방 제국이 돌아가는 뿌듯함
- **주요 요소**:
  - 장인 관리
  - 공급망 구축
  - 던전 길드 관계
  - 연구 트리

## 4. 디자인 레버 (Design Levers)

### 4.1 레시피 계보
| 레버 | 설명 |
|------|------|
| 기본→고급→전설 | 레시피 계보(기본→고급→전설) 구조 설계 |
| 구현 방향 | 철검→강철검→미스릴검→전설검, 하위 마스터 시 상위 해금 |

### 4.2 희귀도-가격 상관
| 레버 | 설명 |
|------|------|
| 경제 연동 | 재료/레시피 희귀도와 시장 가격의 상관 관계 표현 |
| 구현 방향 | 희귀 재료 수급 난이도 → 제작품 가격 상승, 수요-공급 그래프 |

### 4.3 크로스게임 연계
| 레버 | 설명 |
|------|------|
| 아이템 연계 가능성 | 다른 게임/던전과의 아이템 연계 가능성 열어두기 |
| 구현 방향 | game_0003 용병단에 무기 공급, game_0010 상점에 도매 |
