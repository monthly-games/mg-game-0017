## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 0010 데이터 참조 |
| 3~6주 | 레시피 계보 시스템 밸런스 |
| 7~10주 | 고급 제작 튜토리얼 |
| 11~12주 | 첫 시즌 레시피 준비 |

**핵심 과제:**
- 0010 데이터 기반 경제 설계
- 레시피 계보 재미 밸런스
- 고급 제작 진입 장벽 완화

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 시즌별 레시피/소재 테마 1차 |
| 5개월 | 던전 아이템 연계 이벤트 |
| 6개월 | 시즌별 레시피/소재 테마 2차 |
| 7개월 | 제작 랭킹 시스템 |
| 8개월 | 시즌 3차 + 여름 대축제 |

**핵심 과제:**
- 시즌별 테마 레시피 사이클
- 타 타이틀 던전 연계
- 제작 경쟁 요소 추가

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 다른 타이틀 아이템 연계 확대 |
| 10개월 | 제작/경제 도전 모드 |
| 11개월 | 크래프트 챔피언십 예선 |
| 12개월 | 1주년 크래프트 페스티벌 |

**핵심 과제:**
- 크로스 게임 아이템 연계
- 도전 모드 시스템
- 챔피언십 이벤트
