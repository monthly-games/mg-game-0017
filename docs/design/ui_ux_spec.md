# MG-0017: Dungeon Craft UI/UX 스펙

> **게임명**: Dungeon Craft (던전 크래프트)
> **장르**: 제작 던전 RPG
> **카테고리**: Year 2 Core
> **문서 버전**: 1.0.0

---

## 1. 화면 설정

| 항목 | 값 |
|------|-----|
| **방향** | Portrait (세로) |
| **기준 해상도** | 1080 × 1920 |
| **지원 비율** | 16:9 ~ 21:9 |
| **타겟 FPS** | 60fps |

### 방향 선택 이유
- 제작 슬롯 세로 배치
- 던전 탐험 세로 스크롤
- 레시피 목록 세로 스크롤

### 제작 슬롯

| 요소 | 크기 | 설명 |
|------|------|------|
| 슬롯 카드 | 가로 전체 ×80dp | 제작 상태 |
| 프로그레스 바 | 200×12dp | 진행률 |
| 가속 버튼 | 60×36dp | 광고/젬으로 가속 |

---

## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 레시피 선택 | 레시피 목록 | 탭 |
| 제작 시작 | 시작 버튼 | 탭 |
| 가속 | 가속 버튼 | 탭 |
| 완료 수령 | 완료된 슬롯 | 탭 |

### 터치 피드백
- 제작 시작: 슬롯 활성화
- 가속: 프로그레스 점프
- 완료: 아이템 팝 + 파티클
